# GameNightGPT Design Brief

Based on the visual analysis of the provided reference image, this document outlines the core design system for the GameNightGPT project.

## 1. Visual Aesthetic
The overall aesthetic is **Warm, Tactile, and Premium**. It is designed to evoke the feeling of a physical board game experience—cardboard, parchment, and wood—while maintaining a modern, clean digital interface.

*   **Mood**: Cozy, Intellectual, Polished, "Board Game Cafe" vibes.
*   **Textures**: Subtle noise, parchment paper feel, matte finishes.
*   **Depth**: Use of soft, diffuse shadows and layered elements (cards on a table) to create a sense of depth and hierarchy.

## 2. Typography
The typography balances character with high legibility.

### Headings
*   **Font Family**: **Outfit**, **DM Sans**, or **Fraunces** (Soft Serif).
*   **Characteristics**: Rounded geometric sans-serif or warm soft serif. High x-height, open counters.
*   **Usage**: Main headlines, section titles.
*   **Example**: "Instant answers to board game rules questions."

### Body Text
*   **Font Family**: **DM Sans**, **Inter**, or **Nunito**.
*   **Characteristics**: Clean, neutral, high legibility at small sizes.
*   **Usage**: Paragraph text, card content, button labels.
*   **Example**: "Stop flipping through rulebooks mid-game..."

## 3. Color Palette
A warm "Coffee & Parchment" palette.

### Primary Backgrounds
*   **Dark Background (Header/Footer)**: Deep Espresso / Dark Chocolate
    *   Approx Hex: `#261C15` or `#2E201B`
*   **Light Background (Main Content)**: Warm Cream / Parchment
    *   Approx Hex: `#F5F1E8` or `#FAF8F1`

### Accents & UE Elements
*   **Primary Accent (Buttons/Highlights)**: Muted Gold / Biscuit
    *   Approx Hex: `#EAD4AA`
*   **Secondary Accent**: Burnt Orange / Rust
    *   Approx Hex: `#C25E00` or `#D97706`
    *   Used for emphasis or specific game icons.
*   **Card Backgrounds**:
    *   Slightly darker warm beige, approx `#EBE5D5`.

### Text Colors
*   **On Light Backgrounds**: Dark Espresso `#261C15` (High Contrast)
*   **On Dark Backgrounds**: Warm White / Cream `#F5F1E8`

## 4. UI Components

### Cards & Containers
*   **Shape**: Softly rounded corners (border-radius: `12px` - `16px`).
*   **Border**: Subtle or non-existent strokes; definition comes from shadow and contrast.
*   **Shadows**: Soft, diffuse drop shadows to lift elements off the "table".

### Buttons
*   **Style**: Solid, high-contrast.
*   **Shape**: Rounded rectangles.
*   **Behavior**: Subtle lift or brightness change on hover.

### Icons & Imagery
*   **Game Icons**: Hexagonal shapes, mimicking game tiles.
*   **Style**: 3D, beveled edges, detailed illustrations.
*   **Shadows**: Drop shadows to reinforce the physical token look.

## 5. Layout Patterns
*   **Hero Section**: High contrast (Dark background) with clear value proposition.
*   **Cards**: Used for "How it Works" steps and Game selectors.
*   **Spacing**: Generous whitespace (padding) to avoid clutter, similar to a well-organized rulebook.
